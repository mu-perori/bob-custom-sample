# ドキュメントレビュー 出力フォーマット

## フォーマット

チャットに以下のフォーマットで出力する。

```markdown
## ドキュメントレビュー結果

**文字数**: XX文字
**チェック日時**: YYYY-MM-DD HH:MM

### チェック結果一覧

| ID  | チェック項目             | 判定      |
|-----|------------------------|---------|
| 1 | 1項目1行で完結している    | ✅ OK    |
| 2 | 1000文字以内            | ❌ NG    |
| 3 | 実装手順・コード例なし    | ✅ OK    |

### ❌ NG項目

- **2**：文字数が1243文字であり、上限1000文字を超過しています。

```

## 注意事項

- NGが1件もない場合、該当セクションは出力しない。
- NG項目には、判定の根拠を具体的に記述する。
