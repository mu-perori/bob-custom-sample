## .bobディレクトリが見えない場合
.bobディレクトリはBobで開けば見ることができますが、エクスプローラー/Finderでは通常は見えない設定になってることが多いです。
以下の手順で表示することができます。
Windows: エクスプローラーを開く→上部の「表示」タブ→隠しファイルにチェック
Mac: ショートカットで`Command + Shift + .`
